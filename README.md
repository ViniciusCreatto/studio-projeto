# Landing Page Profissional

Uma landing page moderna e responsiva desenvolvida com HTML, CSS e JavaScript puro, featuring carrossel automático, cabeçalho fixo e otimização para SEO.

## 🚀 Funcionalidades

- **Cabeçalho Fixo**: Navegação sempre visível com efeito de scroll
- **Carrossel Automático**: Hero section com slides automáticos e controles manuais
- **Design Responsivo**: Adaptado para desktop, tablet e mobile
- **Otimizado para SEO**: Meta tags, semântica HTML5 e estrutura otimizada
- **Animações Suaves**: Transições e micro-interações
- **Formulários Funcionais**: Contato e newsletter com validação
- **Acessibilidade**: Skip links, ARIA labels e navegação por teclado
- **Performance**: Lazy loading, código otimizado e animações eficientes

## 📁 Estrutura de Pastas

```
landing-page/
├── assets/
│   ├── css/
│   │   └── style.css          # Estilos principais
│   ├── js/

│   ├── images/                # Imagens do site
│   └── fonts/                 # Fontes personalizadas
├── index.html                 # Página principal
└── README.md                  # Documentação
```

## 🚀 Como Usar

1. **Clone o repositório**:
   ```bash
   git clone [repository-url]
   cd landing-page
   ```

2. **Abra o projeto**:
   - Use um servidor local (Live Server, XAMPP, etc.)
   - Ou abra diretamente o `index.html` no navegador

3. **Personalize**:
   - Altere cores no `:root` do CSS
   - Modifique textos no HTML
   - Adicione suas imagens na pasta `assets/images`
   - Configure formulários para seu backend

## 📝 Personalização

### Cores
Edite as variáveis CSS em `assets/css/style.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #10b981;
    /* ... */
}
```

### Conteúdo
Substitua o conteúdo em `index.html`:
- Textos e títulos
- Informações de contato
- Links de redes sociais
- Meta tags

### Imagens
Adicione suas imagens em `assets/images/`:
- Logo da empresa
- Imagens do carrossel
- Fotos da equipe
- Avatares de depoimentos
